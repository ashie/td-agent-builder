You can start / stop Fluentd via launchctl command. - start fluentd % sudo launchctl load /Library/LaunchDaemons/fluentd.plist - stop fluentd % sudo launchctl unload /Library/LaunchDaemons/fluentd.plist If you have a question, please ask it on Fluentd ML: https://groups.google.com/forum/#!forum/fluentd
